# Customer Segmentation Analysis Project

## Overview

This project focuses on analyzing customer behavior and segmenting customers into different groups using machine learning and data analysis techniques. The goal is to help businesses understand customer purchasing patterns and improve decision-making through data-driven insights.

The project includes:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Customer Segmentation
- Data Visualization
- Machine Learning Techniques
- Business Insights Generation

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

---

## Dataset Features

The dataset includes customer-related information such as:

- Customer ID
- Age
- Gender
- Annual Income
- Spending Score
- Purchase History
- Region
- Product Preferences
